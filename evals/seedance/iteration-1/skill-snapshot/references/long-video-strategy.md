# 超长视频分段策略（>15秒）

## 核心原理

Seedance 2.0 单次生成上限 15 秒。超过 15 秒的视频采用**分段生成 + 视频延长拼接**：

1. 第 1 段正常生成（≤15秒）
2. 将第 1 段视频上传为 @视频1，用「视频延长」生成第 2 段
3. 依此类推，逐段拼接

**关键**：视频延长时，选择的生成时长 = 新增部分的时长（延长 5 秒就选 5 秒）。

## 分段规则

1. 按叙事节奏切分，每段 ≤15 秒
2. 每段之间有**画面衔接点**：上段结尾状态 = 下段开始状态
3. 第 1 段正常生成，后续段用 `将@视频1延长Xs`
4. 每段标注属于整体的第几段、承接内容

## 推荐分段方案

| 总时长 | 推荐分段 |
|--------|----------|
| 16-30秒 | 2 段（首段 15s + 延长段） |
| 31-45秒 | 3 段 |
| 46-60秒 | 4 段 |
| >60秒 | 拆分为独立场景分别生成，再用剪辑软件拼接 |

## 完整示例：30秒废墟探险短片

### 第1段（0-15秒）—— 正常生成

**生成时长**：15秒

#### 提示词

```
15秒废墟探险，冷灰蓝色调，cinematic，0-5秒：Aerial Crane Down从废弃工厂
天窗俯拍下降，阳光从破碎玻璃洒入尘埃中形成god rays；6-10秒：Smooth Dolly
Forward推进穿过锈蚀的机械走廊，地上散落的纸张被脚步带起的气流吹动；
11-15秒：Subtle Push In推进至走廊尽头一扇半掩的铁门，门缝中透出微弱暖光，
探险者的手伸向门把手，定格在即将推门的瞬间。
伴随金属嘎吱声、风穿过管道的低鸣和远处水滴声。
```

#### 衔接点

本段结尾画面：探险者的手握住铁门把手，门缝透出暖光，走廊背景阴暗。

---

### 第2段（15-30秒）—— 视频延长

**操作**：将第1段生成的视频上传为 @视频1
**生成时长**：15秒

#### 提示词

```
将@视频1延长15秒。0-5秒：接上段推门画面，铁门被推开，Slow Dolly Forward
缓慢推进，暖光涌入，画面从阴暗过渡到明亮；6-10秒：Gradual Crane Up + Pan
升起并横摇，揭示门后是一个巨大的废弃温室，阳光穿透玻璃穹顶，野生藤蔓爬满
钢架，蝴蝶在光柱中飞舞；11-15秒：Dreamy Orbit缓缓环绕温室中央一棵独自
生长的大树，树冠穿透破碎的穹顶伸向蓝天，音效从寂静过渡为鸟鸣和微风。
```
