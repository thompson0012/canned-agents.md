# Seedance Video Prompt — Baseline (No Skill)

## Task
15秒竖屏治愈风视频：橘猫从沙发醒来，伸懒腰，跳到窗台看雨。

---

## Complete Video Prompt (Chinese)

**画面比例：9:16（竖屏）**

**0-3秒：**
温暖的室内场景，柔和的自然光透过窗帘洒落。一只毛茸茸的橘猫蜷缩在米白色沙发上沉睡，缓缓睁开眼睛，耳朵微微抖动。特写镜头，浅景深，暖色调，慢动作。

**4-7秒：**
橘猫慵懒地抬起头，前爪向前伸展，弓起背部打了一个大大的懒腰，嘴巴微张露出小舌头。中近景，镜头缓慢上移跟随猫咪动作，光线柔和温暖。

**8-11秒：**
橘猫轻盈地从沙发上一跃而起，优雅地跳上木质窗台。跟拍镜头，平滑运镜从沙发过渡到窗台，动作流畅自然。

**12-15秒：**
橘猫端坐在窗台上，透过玻璃窗静静注视外面淅淅沥沥的雨滴。窗外雨丝朦胧，玻璃上滑落的水珠映出柔和光影。镜头从猫咪侧面缓慢推近至半身特写，雨声轻柔，氛围宁静治愈。

**整体风格关键词：** 治愈系，暖色调，胶片质感，浅景深，自然光，慢节奏，温馨氛围

---

## Structure Explanation

### Format Used: Timestamp Storyboard Method (时间戳分镜法)

The prompt is structured into **4 time segments** covering the full 15 seconds:

| Segment | Duration | Content | Purpose |
|---------|----------|---------|---------|
| 0-3s | 3s | Cat sleeping, waking up | Establishing shot, set mood |
| 4-7s | 4s | Stretching | Character action, build warmth |
| 8-11s | 4s | Jumping to windowsill | Transition / movement |
| 12-15s | 4s | Watching rain | Climax moment, emotional payoff |

### Design Decisions

1. **Aspect Ratio**: 9:16 vertical as specified — optimized for mobile/short-video platforms
2. **4 segments** for 15 seconds: Each segment is 3-4 seconds, giving Seedance enough time per shot to render coherent motion
3. **Each segment includes**: Scene description + subject action + camera movement + lighting/mood — the standard elements Seedance needs for quality output
4. **Style keywords appended** at the end to reinforce the overall aesthetic across all segments
5. **Camera language**: Used Chinese cinematography terms (特写, 中近景, 跟拍, 推近) for platform compatibility
6. **Sensory details**: Rain sounds, light reflections, texture descriptions to guide the AI toward richer output
7. **Pacing**: Slow start (sleeping) → gentle action (stretch) → dynamic movement (jump) → calm ending (watching rain) — classic 治愈系 narrative arc
